FSDiscordFormSubmitEvent

Event payload passed into onDiscordFormSubmitCallback for form submissions.

8 methods documented
No entries match that search.

DiscordFormSubmitEvent:getCustomID()

Returns the custom ID for this discord form submit event.

local customID = event:getCustomID()
print("customID:", customID)
DiscordFormSubmitEvent:getPlayer()

Returns the player for this discord form submit event.

local player = event:getPlayer()
print("player:", player)

DiscordFormSubmitEvent:reply(content, optional - flags)

Runs the reply operation for this discord form submit event. It uses content, optional flags.
