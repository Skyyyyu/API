TTile

A single block/tile inside a world — foreground, background, data & flags.

13 methods documented
No entries match that search.

GG · 10 methods

Tile:getTileID()

Returns the tile ID for this tile.

local tileID = tile:getTileID()
print("tileID:", tileID)
Tile:getTileItem()

Returns the tile item for this tile.

local tileItem = tile:getTileItem()
print("tileItem:", tileItem)
Tile:getLastPunched()

Returns the last punched for this tile.

local lastPunched = tile:getLastPunched()
print("lastPunched:", lastPunched)
Tile:getPosX()

Returns the pos x for this tile.

local posX = tile:getPosX()
print("posX:", posX)
Tile:getPosY()

Returns the pos y for this tile.

local posY = tile:getPosY()
print("posY:", posY)
Tile:getNote()

Returns the note for this tile.

local note = tile:getNote()
print("note:", note)
Tile:getTileForeground()

Returns the tile foreground for this tile.

local tileForeground = tile:getTileForeground()
print("tileForeground:", tileForeground)
Tile:getTileBackground()

Returns the tile background for this tile.

local tileBackground = tile:getTileBackground()
print("tileBackground:", tileBackground)
Tile:getTileData(data_type)

Returns the tile data for this tile.

local tileData = tile:getTileData(1)
print("tileData:", tileData)
Tile:getFlags()

Returns the flags for this tile.

local flags = tile:getFlags()
print("flags:", flags)
